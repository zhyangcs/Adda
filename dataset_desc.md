## Dataset Description
In Adda, we include 14 public dataset, include 8 classification datasets and 6 regression datasets from [UCI repository](https://archive.ics.uci.edu/), [OpenML](https://www.openml.org/) and [Kaggle](https://www.kaggle.com/), ranging from 100 to 1M rows. Each dataset was randomly partitioned into an 80\% training set and a 20\% testing set.
